# 人岗匹配与推荐模块

## 模块职责

负责毕业生画像与岗位画像的匹配度计算、岗位推荐排序和技能差距分析。

## 目录说明

| 目录 | 用途 |
|------|------|
| `profile/` | 学生画像和岗位画像构建 |
| `matching/` | 各维度匹配度计算（技能、专业、学历、经验、城市、薪资） |
| `ranking/` | Top-N 排序和相似岗位推荐 |
| `explanation/` | 推荐理由和技能差距分析 |
| `evaluation/` | 推荐效果评估指标 |

## 匹配权重

| 维度 | 权重 |
|------|------|
| 技能匹配度 | 40% |
| 经历匹配度 | 20% |
| 期望岗位方向匹配度 | 15% |
| 学历匹配度 | 10% |
| 城市匹配度 | 10% |
| 薪资匹配度 | 5% |

岗位技能少于 3 个时，技能分按已抽取技能证据量折减，避免仅命中一个稀疏技能的岗位获得过高分数。

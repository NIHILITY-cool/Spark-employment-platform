## 3. 已建立的采集脚本

| 文件 | 作用 | 备注 |
| --- | --- | --- |
| `crawler/raw_fetcher.py` | 通用原始数据采集器 | 保存原始响应和 manifest |
| `crawler/build_ncss_fair_detail_seeds.py` | 从招聘会列表生成详情页 URL 和统计接口 URL | 基于 `fairId` 生成 |

`raw_fetcher.py` 支持参数：

- `--seeds`：种子 URL 文件
- `--output`：原始数据输出目录
- `--max-pages`：最大采集页数
- `--delay`：请求间隔秒数
- `--no-follow`：只抓种子 URL，不跟随页面链接
- `--skip-first`：跳过前 N 个种子 URL，用于续采

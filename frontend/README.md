# 前端模块

## 模块职责

基于 Vue 3 + Vite + Element Plus 的前端，提供学生端和高校端的用户界面。

## 环境安装

```bash
cd frontend
npm install
```

## 启动

```bash
npm run dev
```

## 技术栈

- Vue 3
- Vite
- Element Plus
- Axios
- Vue Router
- Pinia
- ECharts

## 页面结构

### 当前已实现

- 岗位市场搜索、城市/方向筛选和本地收藏状态
- 市场城市与热门技能信号
- 演示学生画像编辑、技能增删和等级维护
- 就业期望维护，保存后自动重算推荐
- Top10 推荐、六维匹配轨道、推荐理由和技能差距
- 桌面与移动端 Playwright 交互测试

### 学生端
- 登录注册
- 个人信息
- 技能管理
- 求职意愿
- 岗位搜索
- 岗位详情
- 岗位推荐
- 技能差距
- 收藏岗位

### 高校端
- 高校总览大屏
- 求职意愿分析
- 学生能力分析
- 岗位市场需求
- 专业岗位匹配
- 专业技能差距
- 就业指导建议
- 培养方向建议

### 管理端
- 岗位数据导入
- 数据质量管理
- 字典管理
